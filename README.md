# SAE 2.01
Groupe : 101\
Membres : Audrey Zerrouk-Ambry, Oumou Tall, Mahima Hossain
## Fonctionnalités ajoutés
### Dynamisme en cas de victoire
Des confettis et un son de victoire se joue lorsqu'une partie est gagné.
### Ajout de mode de jeu
Trois modes de jeu ont été ajouté s'adaptant à la difficulté choisie et avec une gestion des messages de fin :
* Vie limitée : chaque paire de carte retournée qui ne sont pas les mêmes font perdre une vie quand le nombre de vies 
tombe à zéros la partie est perdu. Chaque niveau de difficulté à son nomber de vie attribué (4: 3, 5: 4, 6: 5, 8: 7).
* Temps limité : le joueur / la joueuse doit gagner avant le temps imparti sinon iel a perdu. Chaque niveau de difficulté
à son temps limité attribué (4: 15, 5: 20, 6: 25, 8: 40).
* Mélange à chaque paire : dès qu'une paire est trouvé les cartes non trouvées ce mélange.
### Ajout d'un bouton indice
Dans tous les modes, il y a un bouton indice qui permet de retourner toutes les cartes pendant 1 seconde cependant ce 
bouton ne peut être cliqué qu'une fois.
### Ajout d'un block de règles
Un block contenant toutes les règles du jeu a été ajouté pour permettre aux utilisateurs 
d'avoir une vision plus précise des modes et du bouton indice.
